# /close-day — Итоги дня

Собери итоги рабочего дня:

1. Запусти `git log --since="today" --oneline --all` чтобы увидеть все коммиты за сегодня

2. Для каждого коммита кратко опиши что было сделано (1–2 предложения)

3. Сформируй отчёт в формате:

```
## Итоги дня — [дата]

### Что сделано
- [пункт]

### Ключевые изменения
- [файл/компонент]: [что изменилось и зачем]

### Что осталось на завтра
- [пункт]

### Заметки
- [технический долг, подводные камни, идеи]
```

4. Создай папку `logs/` если не существует:
   ```
   mkdir -p logs
   ```

5. Добавь `logs/` в `.gitignore` если ещё не добавлена

6. Сохрани отчёт в файл `logs/day-YYYY-MM-DD.md` с текущей датой

Тон — деловой, без воды. Факты и результаты.
