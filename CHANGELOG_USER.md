# История обновлений «До встречи»

> Этот файл содержит историю обновлений в формате для пользователей приложения.
> Обновляется автоматически при каждом деплое в production.
> Технический changelog — в CHANGELOG.md

---

## v1.4.0 — 17 апреля 2026

✨ Новый онбординг — при первом запуске бот объясняет как работает сервис и создаёт готовое расписание
✨ Готовое расписание «Свободное время» — можно делиться ссылкой сразу, без настройки
✨ Кнопка «Поделиться в Telegram» — красивое сообщение с ссылкой на бронирование
✨ Создание расписания в боте — кнопочный выбор дней, навигация Назад/Отмена
✨ Кнопка «Создать расписание» открывает мини-приложение на нужном экране
✨ Управление админами через бота — добавление/удаление по ID
✨ Команда /reset — сброс данных для тестирования (только админы)
🎨 Обновлён дизайн админ-панели
🐛 Исправлены ссылки в бета-среде
🐛 Уведомление гостю показывает когда будут напоминания
⚡️ Бот для вернувшегося пользователя показывает актуальные встречи и расписания

## v1.2.0 — 15 апреля 2026

✨ Подключение Google Calendar, Яндекс Календарь и Apple Calendar — занятые слоты блокируются автоматически
✨ Встречи из внешних календарей теперь видны в приложении
✨ Офлайн-встречи с указанием адреса места
✨ Слоты блокируются между всеми расписаниями — двойная запись невозможна
✨ Обновлена навигация бота — Главная, Встречи, Расписания, Профиль
✨ Выбор удобного времени напоминаний о встречах
✨ Утром бот спросит гостя «Встреча в силе?» — вы узнаете, если нет ответа
✨ Связаться с поддержкой можно прямо из профиля
🐛 Исправлены ошибки быстрого добавления встречи
🐛 Устранён сдвиг экрана при открытии приложения
⚡️ При удалении расписания можно выбрать, что делать с активными встречами
🔒 Добавлены Политика конфиденциальности и Условия использования
